# Volumes

This folder contains Kubernetes manifest files demonstrating volume usage. Volumes provide persistent or shared storage for containers in Pods.

## Files

- [HostPath-Manifest.yml](HostPath-Manifest.yml): Pod using HostPath volume.
- [Volume-share.yaml](Volume-share.yaml): Shared volume example.
- [Volume_persistent.yaml](Volume_persistent.yaml): Persistent volume example.

# ConfigMaps and Secrets

This folder contains Kubernetes manifest files for creating ConfigMaps and Secrets. ConfigMaps store non-sensitive configuration data, while Secrets store sensitive data like passwords.

## Files

- [configMap-redis.yaml](configMap-redis.yaml): ConfigMap for Redis configuration.
- [Configmaps&secerets.yaml](Configmaps&secerets.yaml): Combined ConfigMap and Secret examples.
- [Secerts_Pod.yaml](Secerts_Pod.yaml): Pod using Secrets.

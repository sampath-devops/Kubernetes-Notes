# Services

This folder contains Kubernetes manifest files for creating Services. Services provide a stable endpoint to access a set of Pods, enabling load balancing and service discovery.

## Files

- [nginx_nodeportsvc.yaml](nginx_nodeportsvc.yaml): NodePort Service for Nginx.
- [nodeport_service.yaml](nodeport_service.yaml): General NodePort Service.
- [Service_creation.yaml](Service_creation.yaml): Basic Service creation example.
- [service_Galary.yaml](service_Galary.yaml): Service for Gallery app.
- [Service_ironDB.yaml](Service_ironDB.yaml): Service for Iron DB.
- [Simplewebapp_nodeport.yaml](Simplewebapp_nodeport.yaml): NodePort Service for simple web app.

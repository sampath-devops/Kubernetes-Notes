# Deployments

This folder contains Kubernetes manifest files for creating Deployments. Deployments provide declarative updates for Pods and ReplicaSets, managing the lifecycle of applications with features like rolling updates and rollbacks.

## Files

- [Deployment.yaml](Deployment.yaml): Basic Deployment example.
- [Deployment_1.yaml](Deployment_1.yaml): Another basic Deployment.
- [Deployment_full.yaml](Deployment_full.yaml): Full-featured Deployment with multiple configurations.
- [Grafana-deployment.yaml](Grafana-deployment.yaml): Deployment for Grafana monitoring tool.
- [Httpd_deployment.yaml](Httpd_deployment.yaml): Deployment for Apache HTTPD server.
- [nginx_deployment.yaml](nginx_deployment.yaml): Deployment for Nginx web server.
- [redis-deployment.yaml](redis-deployment.yaml): Deployment for Redis database.
- [IronGalaryApp.yaml](IronGalaryApp.yaml): Deployment for Iron Gallery application.
- [IronGalaryDB.yaml](IronGalaryDB.yaml): Deployment for Iron Gallery database.
- [Guest-book-App.yaml](Guest-book-App.yaml): Deployment for Guest Book app frontend.
